# 소개
App은 사용자의 건강 데이터를 기록하고 분석합니다. 이 앱은 다음과 같은 문제를 해결합니다:
- 운동 루틴 관리: 운동 계획 관리
- 식단 기록: 칼로리 및 영양소 섭취 관리
- 운동 정보 제공: 다양한 운동과 자세한 설명 제공

# 기능
- 사용자 데이터 입력: 나이, 키, 체중 저장
- 운동 루틴 관리: 운동 계획
- 식단 기록: 영양 정보를 계획하고 시각화
- 운동 정보 제공: 다양한 운동과 그 효과에 대한 상세 설명
